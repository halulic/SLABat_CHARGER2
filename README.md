Overview

This project is a smart charger for 12V SLA (Sealed Lead-Acid) batteries based on the ESP32 microcontroller.

The system monitors battery voltage, charging current, power, and temperature in real time and displays all important parameters on a TFT display.
Charging is controlled automatically to ensure safe and reliable battery charging.

⸻

Main Features
	•	Smart charging control for 12V SLA batteries
	•	ESP32 microcontroller
	•	Real-time monitoring of:
	•	Battery voltage
	•	Charging current
	•	Power consumption
	•	Temperature
	•	TFT display interface
	•	Automatic charge management
	•	MOSFET-based power control
	•	Protection and status indicators

⸻

Hardware

Main components used:
	•	ESP32 development board
	•	INA219 current/voltage sensors
	•	TFT display (ILI9341 / 3.2”)
	•	MOSFET switching stage
	•	NTC temperature sensor
	•	External power supply
	•	12V SLA battery
